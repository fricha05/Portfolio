Wall-J

Java - Work In Progress

Wall-J est un jeu en vue de dessus développé en Java dans lequel on incarne un petit robot nommé Wall-J, et qui a pour tâche de nettoyer la Terre. Pour cela, il est équipé de bombes à compte à rebours qui font se déplacer les détritus et parcours des niveaux créés par l'utilisateur. Ces niveaux sont constitués de blocs de différents types :
- Les murs (Wall): impénétrables et font office d'obstacles
- Les poubelles (Trashcan): impénétrables et utilisées par Wall-J pour nettoyer
- Les detritus (Garbage): Wall-J doit les jeter dans les poubelles
- Wall-J (Robot): le joueur
Pour compléter un niveau, le joueur doit poser les bombes de manières à ce que lorsqu'elles explosent, il n'y ait plus un detritus dans le niveau.
Le programme permet au joueur de créer ses propres niveaux sous la forme de fichiers texte contenant les caractère W (Wall), R (Robot), G (Garbage) et T (Trashcan).
Chaque niveau est vérifié et défini comme valide ou non.
Dans la version finale du projet, le joueur pourra cliquer sur les cases où il voudra déplacer Wall-J, appuyer sur la barre d'espace pour poser une bombe et lancer son compte à rebours, et appuyer sur entrer pour valider son choix et ainsi mettre Wall-J à l'abris.
Le programme utilise une libraire graphique simple développée par l'université.
